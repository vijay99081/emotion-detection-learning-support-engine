# Emotion Detection & Learning Support Engine

An AI-powered application that detects students' emotions from text using NLP and Deep Learning (BiLSTM & BERT), then provides personalized learning guidance using Google Gemini AI.

## Features
- Emotion Detection
- BiLSTM Model
- BERT Model
- Confidence Score
- Mixed Emotion Detection
- Gemini AI Guidance
- Analytics Dashboard
- History Tracking
- Streamlit Web Interface
